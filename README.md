# 🎬 Netflix Movie Data Analysis

A data analysis project that explores a movie dataset using Python and visualizes trends such as genres, ratings, popularity, vote counts, and release years.

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a movie dataset to discover useful insights through data cleaning, visualization, and statistical analysis.

## 📂 Dataset

- **Dataset Name:** mymoviedb.csv
- **Records:** 9,837 Movies
- **Features Include:**
  - Title
  - Genre
  - Release Date
  - Vote Average
  - Vote Count
  - Popularity
  - Original Language
  - Overview

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## 📊 Analysis Performed

- Data Cleaning
- Missing Value Analysis
- Duplicate Removal
- Genre Distribution
- Top 10 Genres
- Vote Average Analysis
- Popularity Analysis
- Vote Count Analysis
- Movies Released Per Year
- Top Rated Movies
- Language Distribution

## 📈 Visualizations

- Bar Charts
- Histograms
- Line Charts
- Pie Charts
